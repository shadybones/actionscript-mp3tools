Method detail
export()method
public function export(): ByteArray

Returns a ByteArray that is a SWF file containing a class named SoundClass. SoundClass extends Sound.


Example
This example shows how to use export() to create a Sound object from a MP3 loaded using FileReference or URLLoader : 
var swf:ByteArray = myMP3.export();
var loader:Loader = new Loader();
loader.loadBytes(swf);
 //  ... wait for Event.COMPLETE ... 

var MP3Class:Class = Class(loader.contentLoaderInfo.applicationDomain.getDefinition("SoundClass"));
var mySound:Sound = new MP3Class();
mySound.play();

Example
This example shows how to parse an MP3 : 
var myMp3:MP3;
var url:URLLoader = new URLLoader( new URLRequest("myMusic.mp3") );
url.addEventListener(Event.COMPLETE, function(ev){ myMp3 = MP3.parse(ev.target.data);} );
